# Event Backoffice (Next.js)

Cette application permet de piloter l'ensemble des données via une interface web moderne.

## Guide d'utilisation

Pour cloner et lancer cette application, vous devez avoir **Git**, **Node.js** (Version 20.x ou supérieure - LTS recommandée) et **npm** installés sur votre ordinateur.

### Configuration et Lancement

Exécutez les commandes suivantes dans votre terminal pour configurer et démarrer le projet localement :

```bash
# 1. Cloner ce dépôt
$ git clone https://github.com/23karim/Event_backoffice.git

# 2. Accéder au dossier du projet
$ cd Event_backoffice

# 3. Installer les dépendances
$ npm install

# 4. Lancer le serveur de développement
$ npm run dev
```
**Accès à l'application**
Une fois le serveur lancé, ouvrez votre navigateur et accédez à l'adresse suivante :
http://localhost:3000
